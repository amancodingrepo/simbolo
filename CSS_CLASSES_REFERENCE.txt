SPACING & LAYOUT CLASSES
=====================================

SPACING:
- flat-spacing: Default vertical padding/margin for sections
- pt-0: padding-top: 0
- p-0: padding: 0 (all sides)
- mb-24: margin-bottom (24 units)
- mb-10: margin-bottom (10 units)
- mb-8: margin-bottom (8 units)
- lg-mb-24: margin-bottom responsive for large breakpoint
- gap-2: flexbox gap, 2 units
- gap-4: flexbox gap, 4 units
- gap-12: flexbox gap, 12 units
- gap-20: flexbox gap, 20 units
- gap-24: flexbox gap, 24 units

GRID/RESPONSIVE:
- container: Max-width centered container (Bootstrap)
- row: Flex row wrapper
- col-12: Full width (12/12 columns)
- col-md-6: 50% width on medium+ breakpoint
- col-md-4: 33.33% width on medium+ breakpoint
- col-lg-6: 50% width on large+ breakpoint
- col-lg-3: 25% width on large+ breakpoint
- col-lg-4: 33.33% width on large+ breakpoint
- col-xxl-5: Custom large breakpoint, 5 units
- col-xxl-7: Custom large breakpoint, 7 units
- d-grid: CSS Grid display

FLEXBOX:
- d-flex: display: flex
- flex-wrap: flex-wrap
- justify-content-center: justify-content: center
- justify-content-between: justify-content: space-between
- align-items-center: align-items: center
- align-items-end: align-items: flex-end
- align-items-baseline: align-items: baseline

Z-INDEX:
- z-5: z-index: 5 (for layering/positioning)
- position-relative: position: relative

================================
ANIMATION CLASSES
================================

ANIMATION BASE:
- effectFade: Base fade animation class

ANIMATION DIRECTIONS:
- fadeUp: Fade in from bottom
- fadeDown: Fade in from top
- fadeLeft: Fade in from left
- fadeRight: Fade in from right
- fadeZoom: Fade in with zoom
- fadeRotateX: Fade in with X-axis rotation
- fadeRotateY: Fade in with Y-axis rotation

APPLIED TO:
- Applied to content divs with: class="[content] effectFade [direction]"
- Examples:
  - <div class="title text-display-2 effectFade fadeRotateX">
  - <div class="sub fw-semibold effectFade fadeUp">
  - <p class="text effectFade fadeUp">

IMAGE ANIMATIONS:
- img-transform-3: Applied to <img> elements for animation effects
  - Used in: hero section, about section, team cards, etc.

SCROLL ANIMATIONS:
- All animation classes trigger via Intersection Observer
- Typically combined with data-inviewport="yes"

================================
DATA ATTRIBUTES
================================

COUNTER ANIMATION:
- data-speed="[integer]": Animation duration in milliseconds
  - Examples: 1000, 1500, 2000
  - Applied to: <span class="number">

- data-to="[integer]": Target counter value
  - Examples: 120, 500, 1000
  - Applied to: <span class="number">

- data-inviewport="yes": Trigger when visible in viewport
  - Used on: Counter elements
  - Requires: Intersection Observer implementation

CAROUSEL:
- data-clone="[integer]": Number of clone slides for infinite loop
  - Examples: data-clone="5"
  - Applied to: .infiniteSlide elements
  - Purpose: Creates seamless infinite carousel loop
